This repository contains code files for the 12 projects in the course CS50AI.

crossword.py, util.py: AI that solves a given crossword puzzle
degrees.py, generate.py: Search algorithm finding shortest path between two people
heredity.py: Finds the probability of an offspring possessing a certain gene given its parents' genes
puzzle.py, logic.py: Knowledge representation by AI and how it deduces information
minesweeper.py, runner.py: AI agent that plays Minesweeper perfectly
nim.py, play.py: AI agent that plays the game NIM perfectly
pagerank.py: Replicates the Google pagerank algorithm
parser.py: Natural Language Processing
mask.py: Predicts the missing word via recurrent neural networks (RNN)
tictactoe.py, runner.py: AI agent that plays Tic Tac Toe perfectly
shopping.py: Predicts shopping habits based on clustering algorithms
traffic.py: Classifies traffic signs using convolutional neural networks (CNN)
